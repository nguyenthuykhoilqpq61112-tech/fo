# CU Bet — Elite Football & Sportsbook Simulator

CU Bet is an offline-first, professional 32-team knockout football simulation and betting game. It features a real-time match stats engine, interactive single/accumulator betting slips, player development analytics, and VIP luxury store. 

### Core Features
- **Real-Time Match Simulation:** Proprietary tick-by-tick simulation driving possession, goals, fouls, and injuries dynamically based on team composition.
- **Dynamic Bookmaker Engine:** Live-shifting 1X2 and BTTS odds calculations.
- **Player Development:** Sophisticated fatigue, morale, and performance mechanics for a realistic career simulation experience.
- **Elite Fantasy Casino:** Extend the betting experience with Roulette and Crash mini-games.
- **VIP Luxury Store:** Accrue wealth to acquire high-end digital real estate, superyachts, private aviation, and hypercars.

### Getting Started

Install the project dependencies inside the codebase:

```sh
npm install
npm run dev
```

### Architecture
- React TypeScript
- Tailwind CSS
- Frame-locked 2D Canvas visualizations
- Zero external backend dependencies (Offline-First Storage)
